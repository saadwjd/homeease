# 🏠 HomeEase — Trusted Home Services App

A fully functional Flutter mobile app connecting homeowners with verified service professionals in Pakistan.

## 📱 Features
- Google Sign-In & Email Authentication
- Browse & book verified service providers
- Real-time chat with providers
- Live booking notifications
- Provider dashboard with earnings tracker
- Admin dashboard for platform management
- Interactive map with nearby providers
- Rating & review system
- JazzCash & cash payment options

## 🛠 Tech Stack
- **Frontend:** Flutter (iOS & Android)
- **Backend:** Firebase (Auth, Firestore, Storage, Messaging)
- **State Management:** Riverpod
- **Navigation:** Go Router
- **Maps:** Google Maps Flutter

## ⚙️ Setup
1. Clone the repo
2. Run `flutter pub get`
3. Add your own `firebase_options.dart`, `google-services.json` and `GoogleService-Info.plist`
4. Run `flutter run`
